# Repository Structure

Top-level directories:

- `astra/`: compiler and tooling implementation
- `stdlib/`: canonical Astra stdlib sources
- `runtime/`: runtime C assets used by native backend
- `docs/`: project documentation
- `examples/`: runnable Astra examples
- `tests/`: automated test suite
- `editors/vscode/`: VS Code extension and bundled server mirror

Important top-level files:

- `pyproject.toml`
- `Makefile`
- `SPEC.md`
- `INSTALL.md`
