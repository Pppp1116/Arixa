# Astra Reference Manual

## CLI
- `astra build <in> -o <out> [--target py|x86_64]`
- `astra run <in>`
- `astra test`
- `astra selfhost`

## Tooling
- `astpm init/add/lock`
- `astfmt <file>`
- `astlint <file>`
- `astdoc <in> -o <out>`
- `astlsp`
- `astdbg <py script>`
- `astprof <py script>`

## Standard library modules
- collections, io, net, serde, process, time, crypto
