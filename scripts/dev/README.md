# Development Scripts

This directory contains development and debugging scripts used during ASTRA compiler development.

## Contents

### Debug Scripts
- `debug_*.py` - Various debugging utilities for different compiler components
- `debug_enum_*.py` - Enum-related debugging scripts
- `debug_lexer.py`, `debug_lexer2.py` - Lexer debugging
- `debug_parser.py` - Parser debugging
- `debug_semantic.py` - Semantic analysis debugging
- `debug_codegen.py` - Code generation debugging

### Benchmark Scripts
- `benchmark_output_*.py` - Benchmark result outputs
- `compare_debug.py`, `compare_release.py` - Performance comparison scripts

### Status and Test Scripts
- `final_*.py` - Final status reporting and comprehensive test scripts

## Usage

These scripts are primarily used by developers for:
- Debugging specific compiler components
- Running performance benchmarks
- Generating status reports
- Testing new features

## Note

These scripts are not part of the core compiler distribution and are intended for development use only.
